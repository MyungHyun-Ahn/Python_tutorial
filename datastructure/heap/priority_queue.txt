우선 순위 큐
 - 추상 자료형
 - 데이터를 저장할 수 있다.
 - 저정한 데이터가 우선순위대로 나온다.

5, 2, 3, 8 - 데이터를 삽입
8, 7, 5, 2 - 큰 숫자부터 꺼내고 삭제

우선 순위 큐 구현
 - 힙을 이용하면 효율적으로 구현할 수 있다!

 - 힙에 데이터를 삽입
 - 힙 요건을 충족하게 맞추어준다.

힙에서 root 노드 데이터 추출하기
 - root노드와 제일 끝 노드의 자리를 바꾼후
 - 제일 끝 노드의 값을 저장
 - 힙 요건을 충족하게 바꾼후
 - 저장한 값을 리턴

 	                      | 데이터 삽입	| 데이터 추출
     정렬된 동적 배열      |     O(n)   |    O(1)
정렬된 더블리 링크드 리스트 |     O(n)	 |    O(1)
	        힙            |   O(lg(n)) |  O(lg(n))


- 정렬된 동적 배열이나 정렬된 더블리 링크드 리스트를 사용하면 데이터 추출할 때 더 효율적
- 힙을 사용하면 데이터를 삽입할 때 더 효율적
