재귀함수 

자기 자신을 실행하는 함수

재귀적으로 문제를 푼다는 것 = 부분 문제의 답을 이용해서 기존 문제를 푸는 것

활용 예시
팩토리얼
n=0인 경우 n! = 1 : base case
n>0인 경우 n! = (n-1)! * n : recursive case >> 재귀적으로 부분 문제를 푸는 경우

반복문 <-> 재귀함수
반복문도 가능하면 재귀함수도 가능하다.

재귀함수가 너무 쌓이면 callstack이 많이 쌓이면 stackoverflow 에러
python은 callstack 1000으로 제한

반복문을 쓰는 게 더 깔끔하거나 효율적일 때가 있고
재귀 함수를 쓰는 게 더 깔끔하거나 효율적일 때가 있다.
