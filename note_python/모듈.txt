모듈

여러가지 기능을 정리해 둔 파이썬 파일

여러 요소를 한 파일에 구현하면 복잡해짐
어느 부분이 어떤 역할을 하는지
기능을 수정하고 싶으면 어떤 부분을 수정해야 하는지

코드를 가져와서 재사용 가능

__이름__ : 특수 변수 > 던더 라고 읽음

파이썬이 내부적으로 관리하는 변수

네임스페이스

파일에서 정의된 모든 이름 
dir() 함수는 파일의 네임스페이스를 리턴

파이썬에서는 똑같은 이름의 함수가 정의되면 가장 마지막의 함수가 사용됨

스탠다드 라이브러리

int, float, string 같은 자료형
print, dir 같은 내장 함수
유용한 기능을 제공하는 모듈들 (스탠다드 모듈)

모듈 검색 경로

import sys
sys.path >> 모듈이 있는 경로를 리턴

sys.path.append() >> 경로를 새로 추가함 >> 다만 프로그램을 종료하면 초기화 됨

스크립트 >> 실행 파일
프로그램을 작동시키는 코드를 담은 실행 용도의 파일

모듈 >> 불러오는 파일
프로그램에 필요한 변수들이나 함수들을 정의해 놓은 파일
