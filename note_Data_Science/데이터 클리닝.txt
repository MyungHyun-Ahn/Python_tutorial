데이터 클리닝 - 좋은 데이터를 만드는 방법

1. 완결성 2. 유일성 3. 통일성

3가지 조건을 갖추어야 좋은 데이터

1. 완결성
결측값(NaN)을 찾고 그것을 제거하거나 대체

isnull() : 결측값의 유무에 따라 True 와 False로 반환하는 메서드
df.isnull().sum() 형식으로 결측값의 개수를 카운트 가능

dropna() : 결측값을 가진 행을 지워주는 메서드 
axis="columns" 로 칼럼을 지우는 방법도 있음

fillna() : 결측값을 어떠한 값으로 채워주는 메서드 
median() 메서드로 중간값으로 채워주거나 mean() 메서드로 평균값으로 채울 수 있음

2. 유일성
중복되는 데이터를 찾고 삭제

df.index.value_counts() 로 중복되는 데이터를 찾기

중복되는 행이 있다면
drop_duplicate() : 중복되는 행 한개를 지워줄 수 있음
데이터 프레임의 내용을 기준으로 중복 여부를 판단
중복되는 행만 지워줄 수 있음

df.T 로 행과 열을 바꾸어 줄 수 있음

df.T.drop_duplicate().T : 행과 열을 바꾸어준 뒤 중복되는 행을 삭제한 후 다시 되돌려준 데이터 프레임을 저장

3. 통일성

박스 플롯 그래프를 보고 q3 + IQR * 1.5 보다 크고 q1 - IQR * 1.5 보다 작은 범위에 있을 경우 이상점이라 판단
q1은 하위 25%, q3는 상위 75%

quantile(a) :  a인 지점을 구함
